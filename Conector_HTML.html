¿Qué es HTML?

HTML(HyperText Markup Language) es el lenguaje de marcado fundamental que define la estructura básica y el contenido de toda página web. Utiliza un sistema de etiquetas para organizar y jerarquizar elementos como títulos, párrafos, imágenes, enlaces y listas. No es un lenguaje de programación, sino que se enfoca en establecer el esqueleto semántico del contenido, permitiendo que navegadores y otros dispositivos interpreten correctamente la información.

HTML constituye la base estructural indispensable sobre la cual se aplican después de los estilos con CSS y la interactividad con JavaScript. Su correcta implementación es crucial para la accesibilidad, el posicionamiento en buscadores (SEO) y garantizar que el contenido sea interpretable tanto para usuarios como para sistemas automatizados.
¿Para que sirve?

HTML sirve para crear y estructurar el contenido de las páginas web. Es el esqueleto que define todos los elementos básicos como textos, imágenes, enlaces, formularios y secciones. Sin HTML, los navegadores no podrían mostrar ningún contenido web organizado.

Además, proporciona la base semántica esencial sobre la cual se aplican posteriormente los estilos visuales mediante CSS y la interactividad dinámica a través de JavaScript.
20 etiquetas pupulares:

1. `< !DOCTYPE >` - Declara el tipo de documento
2. `< html >` - Contenedor raíz del documento
3. `< head >` - Metadatos del documento
4. `< title >` - Título de la página
5. `< body >` - Cuerpo del documento
6. `< h1 >` - Encabezado
7. `< p >` - Párrafo
8. `< a >` - Enlace
9. `< img >` - Imagen
10. `< div >` - División o contenedor
11. `< span >` - Contenedor en línea
12. `< ul >` - Lista desordenada
13. `< ol >` - Lista ordenada
14. `< li >` - Elemento de lista
15. `< table >` - Tabla
16. `< form >` - Formulario
17. `< input >` - Campo de entrada
18. `< button >` - Botón
19. `< nav >` - Navegación
20. `< footer >` - Pie de página

Estas etiquetas forman la base estructural y semántica de cualquier documento HTML.
Su estructrura básica es la siguiente:

< !DOCTYPE html>
< html>
< head>
< title> < /title>
< /head>
< body>
(Contenido)
< /body>
< /html>
Cambios Historicos:

HTML ha experimentado una notable evolución desde su creación en 1991 por Tim Berners-Lee, cuando surgió como un lenguaje simple con apenas 18 etiquetas destinado principalmente a compartir documentos académicos. En 1995, HTML 2.0 se estableció como el primer estándar oficial, introduciendo elementos cruciales como formularios, ademas HTML 3.2 expandió las capacidades incorporando tablas complejas, applets de Java y scripts.
